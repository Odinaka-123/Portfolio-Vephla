## Overview

This is a **personal portfolio website** designed to highlight my skills, projects, work experiences, and provide a way for clients or employers to contact me. The website is **mobile-first, responsive, and visually appealing**.  

**Live Preview:** *https://portfolio-vephla.vercel.app*

---

## Features

- Responsive navigation bar with a logo and resume download button  
- Hero section with an illustration and animated social links  
- Skills section with interactive skill cards  
- Experience section with work history cards  
- About section highlighting personal details and profile image  
- Projects section with featured projects and live/demo links  
- Testimonials section for client feedback  
- Footer with social media links  
- Fully responsive across devices  

---

## Technologies Used

The project is built using:

- **HTML5** – Markup structure of the website  
- **CSS3** – Styling and layout, including Flexbox and Grid  
- **Font Awesome** – For icons (e.g., social links, skill icons)  
- **Google Fonts** – For typography
